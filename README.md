# 🚀 Aadhikrish Portfolio (MERN Stack)

A full-stack **developer portfolio + CMS platform** built using the MERN stack.  
It showcases projects, blogs, and skills with a **custom admin dashboard** for dynamic content management.

> 🔗 GitHub Repo: https://github.com/Aadhikrish23/Aadhikrish-Portfolio.git

---

## ✨ Overview

This project is more than just a portfolio — it is a **custom-built CMS (Content Management System)** that allows dynamic control over portfolio content.

- Manage blogs, projects, and skills
- Upload and manage images via Cloudinary
- Secure admin dashboard for content control
- Dynamic frontend powered by APIs

---

## 🔥 Features

### 🌐 Public Portfolio
- Responsive and modern UI (React + Tailwind CSS)
- Dynamic Projects, Blogs, and Skills sections
- Blog listing and detailed blog pages
- Clean and reusable component architecture

### 🔐 Admin CMS (In Progress)
- JWT-based authentication system
- Protected admin routes
- Dashboard to:
  - Create / Edit / Delete Blogs
  - Manage Projects
  - Manage Skills
- Image upload with Cloudinary integration

### ⚙️ Backend
- RESTful API architecture
- MVC pattern for scalability
- Input validation and centralized error handling
- Slug generation for blog URLs
- Service-based modular structure
